# FastApi do Zero 2025
Curso Dinossauro - Eduardo Mendes 2025

O curso FastApi do Zero é um curso completo de FastAPI, onde você aprenderá a criar APIs RESTful de forma rápida e eficiente utilizando o framework FastAPI. O curso é voltado para desenvolvedores que desejam aprender a criar APIs modernas e escaláveis com Python.
